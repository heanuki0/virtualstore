## 핵심 논의 사항 (5대 주제)

1. **전체 UX 플로우** — 롯데 앱 → 콘란샵 탭 → 로비 → 쇼룸(3D) → 상품 탐색 → 구매 페이지 연결
2. **전시형 vs 커스터마이징형** — 온라인용 고퀄 전시형 + 매장 태블릿용 배치형 병행 가능 여부
3. **로비 컨셉 결정** — 헤리티지/앤틱 vs 모던 방향성 (→ **모던 확정**)
4. **AI 어시스턴트 기능** — 스테디셀러 추천, Q&A, LLM 학습 범위/제한사항
5. **앱 내 임베딩 vs 외부 리디렉션** — 앱 이탈 없는 구현 방식

## 칼리버스 측 제안

- **외관 생략, 로비에서 시작** — "외관은 고비용, 사용자 경험도 많지 않으니 로비에서 시작"
- **프리셋 기반 전환** — 개별 가구 배치 대신 미리 만든 세팅 3~4가지 탭 전환
- **기술적 한계 투명 공개** — "웹 베이스에서 다수 3D 모델 로드 시 퀄리티 저하 불가피"
- **3D 파노라마 웨이포인트 방식** — 전체 조망 + 상품 근접 포인트 2~3곳
- **AI 모델링 대안** — 매장 태블릿용은 포토리얼 대신 AI 모델링으로 레이아웃만
