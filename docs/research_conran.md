# 테렌스 콘란 경 & 더 콘란샵 리서치 문서

> 콘란샵 웹 버추얼 스토어 외관 디자인 방향성 수립을 위한 조사 자료

---

## 1. 테렌스 콘란 경 (Sir Terence Conran, 1931–2020)

### 생애
- **본명:** Terence Orby Conran CH RDI FCSD
- **출생:** 1931년 10월 4일, 영국 서리 킹스턴
- **서거:** 2020년 9월 12일 (향년 88세)
- **학력:** Central School of Art and Design (텍스타일 전공, 바우하우스+Arts & Crafts 영향)
- **작위:** 1983년 기사 작위(Knight Bachelor) 서훈

### 핵심 경력
| 연도 | 업적 |
|------|------|
| 1956 | Conran Design Group 설립 |
| 1964 | **Habitat** 1호점 오픈 (런던 풀럼 로드) — 모던 가구 대중화의 시작 |
| 1973 | **The Conran Shop** 1호점 오픈 |
| 1987 | 미슐랭 빌딩 매입 → Bibendum 레스토랑 + 콘란샵 플래그십 |
| 1989 | **Design Museum** 설립 (런던 셰드 템즈) |
| 2000s | 50개 이상의 레스토랑 운영 (Bibendum, Bluebird, Quaglino's 등) |
| 저서 | 50권 이상, 전 세계 2,500만부 판매 |

## 2. 디자인 철학: "Plain, Simple, Useful"

### 핵심 원칙

> **"디자이너의 일은 세상이 어떤 모습인지가 아니라, 어떤 모습이어야 하는지를 상상하는 것이다."**
> — Sir Terence Conran

- **Plain (소박함):** 불필요한 장식을 배제한 정직한 형태
- **Simple (단순함):** 조립하기 쉽고 어디에나 어울리는 보편적 디자인
- **Useful (유용함):** 일상의 질을 높이는 실용적 기능

### 주요 어록

| 어록 | 맥락 |
|------|------|
| "좋은 디자인은 98%가 상식이고, 2%가 미학이다." | 디자인의 본질에 대해 |
| "가구와 음식은 사람이 삶에 대한 태도를 정의하는 방식이다." | 라이프스타일 철학 |
| "좋은 디자인을 믿는 것은 신을 믿는 것과 같다. 당신을 낙관주의자로 만든다." | 디자인과 삶의 관계 |
| "합리적이고 지적인 사람들에게 잘 만들고 잘 디자인한 것을 적정 가격에 제공하면, 그들은 그것을 좋아하고 살 것이다." | 대중을 위한 디자인 |

### 미학적 영향
- **바우하우스:** 기능주의, 산업 디자인과 예술의 통합
- **이탈리안 디자인:** Gio Ponti, Piero Fornasetti의 대륙적 감성과 시간을 초월한 우아함
- **프렌치 마켓:** 프로방스 벼룩시장, 노변 카페, 소박하지만 풍요로운 진열에서 영감
- **영국 Arts & Crafts:** 윌리엄 모리스의 "모든 사람에게 좋은 디자인을" 정신

## 3. 더 콘란샵 (The Conran Shop)

### 역사
- **1973년 설립** — 콘란 경이 직접 큐레이션한 라이프스타일 편집숍
- 가구, 조명, 키친웨어, 텍스타일, 향초, 서적까지 "생활 전체를 디자인"하는 토탈 라이프스타일
- 2023년 **50주년** 기념 리브랜딩 (Pentagram 담당)

### 브랜드 아이덴티티 (2023 리브랜딩)
- **로고:** "The Conran Shop" → T.C.S. 축약형 심볼 + 풀네임 병기
- **전용 서체:** Legacy Typeface Collection (바우하우스 감성의 맞춤 서체, Pentagram Sascha Lobe 디자인)
- **톤:** 모더니티 + 정교함 + 신뢰감, 창립자의 비전에 경의
- **컬러:** 블랙/크림 화이트/골드 기반, 따뜻한 뉴트럴

### 글로벌 매장 (12개, 4개국)
| 국가 | 매장 |
|------|------|
| 영국 | Chelsea (미슐랭 빌딩 플래그십), Marylebone, Selfridges |
| 프랑스 | Paris Rive Gauche, Paris Marais |
| 일본 | Tokyo Marunouchi, Shinjuku, Nagoya, Fukuoka |
| **한국** | **롯데백화점 강남점 (세계 최대 규모)**, 잠실점 (2025 오픈) |

## 4. 미슐랭 빌딩 — 콘란샵 런던 플래그십

### 건축 개요
- **위치:** 81 Fulham Road, Chelsea, London SW3
- **원래 용도:** 미슐랭 타이어 영국 본사 (1911년 건축)
- **건축가:** François Espinasse
- **양식:** 아르누보 / 프로토-아르데코 / 분리파 기능주의
- **1985년** 콘란 경 + 출판인 Paul Hamlyn이 매입, 1987년 재개장

### 외관 특징 (가상 스토어 외관 참고)
- **파사드:** 초록+흰색 타일 장식, 기하학적 패턴
- **스테인드글라스:** 미슐랭 광고를 바탕으로 한 대형 스테인드글라스 창 3개
- **큐폴라:** 지붕 양쪽에 타이어 형태의 돔 장식, 야간 조명
- **바닥 타일:** 거리 레벨에 당대 레이싱카를 묘사한 장식 타일
- **구조:** 영국 최초의 철근콘크리트 건축 중 하나 (에네비크 공법)
- **전체 인상:** 산업시대의 기능주의 + 아르누보의 장식미가 공존

## 5. 바튼 코트 — 콘란 경의 자택

### 건물
- **위치:** Kintbury, West Berkshire
- **건축:** 1772년 (조지안 양식)
- **구조:** 붉은 벽돌 5베이 + 중앙 페디먼트 현관
- **등급:** Grade 2 Listed Building
- **거주:** 1971년부터 콘란 경의 주거지

### 인테리어 특징 (콘란 스타일의 정수)
- **거실:** 밝고 통풍 좋은 공간, 흰 배경 + 모던 가구, FLOS Arco 플로어 램프
- **현관:** 프로방스 벼룩시장에서 구입한 프렌치 데코 메탈 테이블 위 유리 컬렉션
- **전체 톤:** 역사적 건물 + 현대적 가구의 조화, 니켈 도어 핸들, 디머 스위치
- **색감:** 크림 화이트 / 자연광 / 따뜻한 목재 / 앤티크 브라스

## 6. 콘란샵 강남 (롯데백화점)

### 매장 개요
- **오픈:** 2019년 11월
- **위치:** 롯데백화점 강남점 신관
- **규모:** 2개 층, 3,305㎡ (약 1,000평) — **전 세계 콘란샵 중 최대 규모**
- **위상:** 4번째 국가, 12번째 글로벌 매장

### 공간 구성
| 층 | 컨셉 | 구성 |
|----|------|------|
| **1층** | **LAB-like** | 향수·향초·소품, 주방용품, 스피커, 욕실용품, 미술품, **콘란 카페** |
| **2층** | **CLUB-like** | 프리미엄 가구, 디자인 조명, 다크 톤 인테리어 |

### 특징
- 인테리어와 제품 색상을 층별로 통일 (1층: 화이트 톤, 2층: 다크 톤)
- 인테리어 전문가 상주 → 맞춤 컨설팅 서비스
- 최초의 "로컬 맞춤형 매장" 컨셉으로 한국 시장 특화

## 7. 가상 스토어 외관 디자인 참고 포인트

### 콘란 스타일의 건축적 키워드
| 요소 | 설명 |
|------|------|
| **재질** | 크림색 석재 + 황동(브라스) 트림 + 폴리시드 대리석 |
| **창문** | 대형 아치형 유리창 (내부가 보이는 쇼윈도) |
| **조명** | 따뜻한 자연광 + 황동 펜던트, 야간 은은한 글로우 |
| **식물** | 입구 양옆 대형 토피어리 / 올리브 나무 |
| **바닥** | 기하학 패턴 대리석 인레이 |
| **색감** | 블랙 #0a0a0a / 크림 #faf8f3 / 골드 #b8935a / 액센트 오렌지 #c85a2a |
| **서체** | 세리프 헤드라인 + 산세리프 본문 (Legacy Typeface 참고) |
| **무드** | "소박하지만 격조 있는" — 영국 조지안 하우스의 정제된 우아함 |

### 미슐랭 빌딩에서 차용할 수 있는 요소
- 파사드의 타일 장식 → 콘란 브랜드 문양으로 치환
- 아르누보 곡선 + 모던 기능주의 직선의 공존
- 대형 스테인드글라스 → 룸셋 프리뷰 역할의 대형 쇼윈도

### 바튼 코트에서 차용할 수 있는 요소
- 붉은 벽돌 + 크림 석재 페디먼트 현관
- 대칭적 5베이 구성 (좌우 대칭 창문)
- 정원 진입로 + 문 앞 계단
- 오후 햇살이 쏟아지는 따뜻한 분위기
